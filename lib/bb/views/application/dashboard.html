Select your workspace

  • Login

    Login workspace enables a workstation to process only Sample logins. Select this option when the workstation does not have any testing devices attached, or when the workstation is only logging Samples received by the lab.

  • Testing

    The Testing workspace enables a workstation to conduct measurement capture for tests where a testing device is connected to your workstation.

  • Flow Line

    The Flowline workspace enables a workstation to process both a Sample login and testing for samples where devices are currently connected to the workstation.
